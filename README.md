## Portfolio-website
A personal portfolio website built using HTML, CSS, and JavaScript to showcase projects, skills, and personal information in a clean and responsive layout.
## Repository Structure
```
portfolio-website/
│
├── assets/
│   ├── css/
│   │   └── styles.css      
│   │
│   ├── img/                
│   │
│   ├── js/
│   │   └── script.js      
│
├── index.html              
└── README.md               
```

## Features

- Responsive design for desktop and mobile
- Clean and minimal UI
- Easy to customize sections
- Organized asset structure
- Beginner-friendly codebase

 ## Tech Stack
  - HTML5 – Structure
  - CSS3 – Styling & layout
  - JavaScript – Interactivity

   ## Customization
   - Edit content →```index.html```
   - Update styles →```assets/css/styles.css```
   - Add images →```assets/img/```
   - Scripts →```assets/js/```
